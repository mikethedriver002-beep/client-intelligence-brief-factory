# Client Onboarding Checklist

## Client profile

- Agency name
- Client brand name
- Client vertical
- Geography
- Competitors
- Topics to watch
- Topics to ignore
- Preferred tone
- Delivery cadence

## Source setup

- Official/client-owned sources
- Competitor websites
- Public databases
- RSS/news sources
- Manual analyst sources
- Prohibited/login-gated sources

## Brief setup

- Top item categories
- Minimum useful items
- QA strictness
- Preferred sections
- Who reviews before client delivery
